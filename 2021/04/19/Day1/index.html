Go-Day1

Day1

Hello world

fmt.Println() is the output function in go, using it by import the package “fmt”

package “math/rand” allow user to use method start with “rand”

If a name starts with a capital letter, then it is exported, meaning that you can use it.

Any “unexported” names are not accessible outside the package.

In this case math. Pi is an exported value.

Function

When the named parameter types of two or more consecutive functions are the same, the last type can be omitted.

It is noticeable that, unlike Java, function in go can return multiple values.

a, b := is also different from other types of assignment statement

Parameter and variable

  • Define a variable

    Variable declarations can contain initial values, one for each variable.

    If the initial value already exists, you can omit the type; the variable will get the type from the initial value.

  • Concise assignment statement :=

    It can can replace var declarations where the type is clear. But it cannot be used outside a function

  • Basic type in Go

    Like import statements, variable declarations can also be “grouped” into a syntax block.

  • Zero value

    The default value for different types of variable

    1. string: “”
    2. bool: false
    3. int, float64: 0

  • Variable conversion

    Go requires explicit conversion when assigning values between different types of items

  • Constant

    Define a constant using const

    Constant cannot be declared by :=

    Numeric constants are with high precision

Basic structure and control statement

for

for statement in Go is different from other languages like Java, C and JavaScript, for that it does not have parentheses around its conditions

An interesting fact is that while statement in C is also for in Go

If

If statement is similar to for, it doesn’t have parentheses.

If statement can declare a variable like for statement

the variable declared in If statement is only valid in if and else if, else statement of it

switch

switch in Go doesn’t require a break statement to end it, unless end with fallthrough statement

Another important difference of Go is that the case of the switch does not need to be a constant, and the value does not need to be an integer.

defer

The defer statement will postpone the function until the outer function returns.

The parameters of the delayed function will be evaluated immediately, but the function will not be called until the outer function returns.

Delayed function calls will be pushed onto a stack. When the outer function returns, the deferred function will be called in the order of LIFO.
