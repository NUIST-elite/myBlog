Day07

The time package provides functions for displaying and measuring time. The calculation of the calendar uses the Gregorian calendar

Time object

use time.Time to acquire time.

 

Timestamp

use time.Unix() to turn timestamp into time object

Manipulate time

Go offers lots of methods to operate on time :)

  • Add

  • Sub

  • Equal

  • Before & After

    A.Before(B): if A earlier than B, return true, else return false

    A.After(B): if A later than B, return true, else return false

 

Timer

Use time.Tick (time interval) to set the timer. The timer is essentially a channel
